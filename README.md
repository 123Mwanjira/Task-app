# Task App 

A **lightweight, interactive web-based task manager** that helps users create, view, and delete tasks easily. Built with **HTML, CSS, and JavaScript**, this app is designed for beginners and showcases real, functional code.

---

## Features 

- **Add tasks with a date** – quickly log what you need to do  
- **Delete tasks interactively** – remove tasks with a single click  
- **Beginner-friendly code** – easy to read, understand, and extend  
- **Ready for enhancements** – can be upgraded with persistent storage, styling, and collaborative features  

---

## Future Plans 

- **Persistent storage** – tasks remain after page refresh using localStorage or backend  
- **Polished UI/UX** – improve visuals, layout, and mobile responsiveness  
- **Collaboration features** – share tasks, assign priorities, real-time updates  
- **Hackathon-ready presentation** – interactive, visually appealing, and professional
