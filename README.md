# Task App 🚀

A simple interactive web-based task manager that helps users create and delete tasks with ease. Built with HTML, CSS, and JavaScript.

## Features
- Add tasks with a due date
- Delete tasks interactively
- Beginner-friendly code, ready for improvement
- Can be extended with persistent storage and styling

## Future Plans
- Save tasks even after refreshing (localStorage)
- Mobile-friendly and responsive design
- Collaborative features for teams
- Hackathon-ready polished UI
